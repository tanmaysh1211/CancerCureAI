# 🎗️ CanCure AI — Breast Cancer Detection

A machine learning web application that predicts whether a breast tumor
is **Malignant** or **Benign** using a Random Forest classifier trained
on the Wisconsin Breast Cancer Dataset.

---

## 📊 Model Performance

| Metric    | Score |
|-----------|-------|
| Accuracy  | 96%   |
| Precision | 93%   |
| Recall    | 98%   |
| F1 Score  | 95%   |
